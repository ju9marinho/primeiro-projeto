As tags tem abertura e fechamento, 
mas algumas somente de abertura.
As tags tem conteúdo ou não, e algumas tags tem 
parametros.
As tags são sempre com letra minúscula.
-----------
Tag de input não tem fechamento.
Tag sem fechamento, o VS Code não completa/fecha.
Tag de imagem é só assim <img>.
Caso eu esqueça ou não saiba qual a tag usar, 
buscar no google.
<img src="colar  image address">.
--------------
Como preparar o HTML:
1 - Todo html tem que começar com <!DOCTYPE html>. 
<!-- define a versão do HTML. -->

2 - Toda pagina precisa conter as tags de abertura e 
fechamento de HTML, é tudo que fica dentro
do site:
<html
</html>

3 - é necessário conter o idioma do site:
lang"pt-br">

4 - <head> Vai as configurações do site.
Por exemplo, o título:
<title> É onde vai o título na barra de endereço.

5 - <body> Colocar todas as tags que
aparecem no site. </body>

6 - DICA - coloca html e o vs code ja traz
as informações acima.
